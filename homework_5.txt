1. Генератор имени

По кнопке выдавать рандомное Имя.

Реализация через файлы.
Например, в файле cats.txt будут клички котов, когда пользователь выбирает Cats, то из этого списка выдается Имя.

2. Добавить в генератор рандомного изображения Имя

3. Телеграм бот для обучения.

Задачи:
    - новый репозиторий;
    - получить токен от BotFather;
    - настростроить виртуальное окружение:
      - создать файл requirements.txt;
      - добавить необходимые библиотеки;
    - создать файл main.py;
    - в main.py добавить базовую структуру бота;
    - протестировать что все работает.
